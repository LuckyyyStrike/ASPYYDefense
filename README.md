# About
Simple tower defense game, meant to be a proof of concept and an exercise in python.

Short list of features I am planning to add

- Player is able to place towers freely
- Different types of towers with range, DPS
- Different kind of enemies with HP, velocity
- Help page to explain controls to the Player

# How to build
Run main.py with Python3.
