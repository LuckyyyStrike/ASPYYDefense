# About
Simple tower defense game, meant to be a proof of concept and an exercise in python.

## Implemented features
- Draws a map in terminal,
- Maps can be defined in *.txt files. Have to be put in the 'maps' folder.

## Planned features
- Enemies spawn in a given rate and follow a given path.
- Towers attack enemies in a given radius.
- Player is able to place towers freely at the start of the game.
- Help page to explain controls to the Player.
- Implement proper gameplay loop with rounds in between waves where player can place towers and a lose condition with game over screen.
- Economy system where placing towers costs resources and surviving rounds gives resources.
- Dynamic width and height for game area.
